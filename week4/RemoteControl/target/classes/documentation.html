Remote Control Rest API Documentation

/remotes

GET /remotes/remote

This returns all the brands of the remotes

GET /remotes/remote/{brand}

This returns all of the remotes for a brand

GET /remotes/listRemotes

This returns the active remotes

GET /remotes/sendCommand/{remote}

This asks for the "buttons that can be pressed"

PUT /remotes/sendCommand/{remote}/{Command}

This sends an actual command to "push a button"

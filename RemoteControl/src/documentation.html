Remote Control Rest API Documentation

/remotedb

GET /remotedb/remote

This returns all the brands of the remotes

GET /remotedb/remote/{brand}

This returns all of the remotes for a brand

/lirc

GET /lirc/remotes

This checks for the actual loaded remotes"

PUT /lirc/remotes/{brand}

this loads the brands remote into lirc"

GET /lirc/{remote}

This asks for the "buttons that can be pressed"

GET /lirc/{remote}/{Command}

This sends an actual command to "push a button"

/Users

GET /Users

Returns all of the users

PUT /Users

Put in a new user

GET /Users/macros/{userId}

Gets the macros for the user id

GET /Users/macros/{userId}/{macroId}

Gets the requested macro for the user id

PUT /Users/macros

Adds to the macro for the user id

GET /Users/favorites

Gets all user favorites

PUT /Users/favorites

Add a user Favorite

GET /Users/favorites/{userId}

see a particular user favorite
